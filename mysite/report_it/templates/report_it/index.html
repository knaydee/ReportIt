Welcome to Report It

What Is Report It?

Report It provides an easy way to log incidents of bullying or sexual harassment.

Why Is This Important?

When certain incidents or situations make you feel uncomfortable, it can be difficult to recognize them as a pattern. Report It can help you track incidents to create a comprehensive report that demonstrates a pattern of bullying or sexual harassment. This can be a starting point to make a solid case and take it further.
